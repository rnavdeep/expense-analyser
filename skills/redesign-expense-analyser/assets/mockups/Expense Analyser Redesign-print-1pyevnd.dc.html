Key cross-page interactions
A
Login → Home. Successful auth redirects home and unlocks the nav.
B
Upload → Textract → Results. A new expense's receipt is processed, then appears in Doc Results.
C
Assign friend → Shared. Assigning a user makes the expense show in their Shared Expenses.
D
SignalR → Notifications. Processing & shares push live; the bell badge updates in real time.
E
Session lost → Login. Any protected route without a valid session bounces back to Login.
